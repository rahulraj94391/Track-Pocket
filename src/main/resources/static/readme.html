Personal Finance Tracking App

A simple web-based application for tracking personal finances, including income, expenses, and budgeting.

Features

  • Add, view, edit, and delete transactions (income and expenses).
  • Organize transactions by categories (e.g., groceries, salary, rent).
  • Set and manage monthly budgets for different categories.
  • View transaction history and track financial trends.
  • Visualize data through reports and charts.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Java (Spring Boot), Node.js (Express), Python (Flask/Django) - select any based on preference
  • Database: MySQL, PostgreSQL, or MongoDB

Getting Started

Prerequisites

  • Java Development Kit (JDK) installed (if using Java/Spring Boot)
  • Node.js installed (if using Node.js/Express)
  • Python installed (if using Python/Flask or Django)
  • Database setup (MySQL, PostgreSQL, or MongoDB)

Setup Instructions

  1. Clone the repository:
    2. git clone https://github.com/your-username/finance-tracker.git
  2. Navigate to the project directory:
    3. cd finance-tracker
  3. Set up the backend server:
    • If using Java/Spring Boot: ./mvnw spring-boot:run
    • If using Node.js/Express: npm install && npm start
    • If using Python/Flask: pip install -r requirements.txt && python app.py
  4. Set up the database:
    • Configure the database connection settings in the backend configuration file (application.properties, config.js, settings.py, etc.)
    • Run database migrations if needed.
  5. Open the application in your web browser at http://localhost:8080 (or the port specified in your configuration).

Usage

Once the app is running, you can perform the following actions:

  • Add new transactions (income/expenses) using the provided form.
  • View the transaction history and filter by category or date.
  • Set budget limits for different categories.
  • Monitor your financial health through reports and visualizations.
